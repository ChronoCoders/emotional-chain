Why This Happens:

Inline style={} can only define base styles
CSS :hover and :focus pseudo-classes in your stylesheet override the inline text color
The text becomes invisible when it changes to the same color as the background
Tailwind classes include proper hover/focus states that prevent this

The key is using hover:text-black and focus:text-black to explicitly maintain text visibility in all states.RetryClaude can make mistakes. Please double-check responses.
