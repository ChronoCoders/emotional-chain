## Validator Rewards

### Base Reward
50 EMO per block (~8,640 blocks/day)

### Performance Bonuses
| Bonus | Condition | Amount |
|-------|-----------|--------|

| Uptime | 100% available | +5% |
| Accuracy | Unanimous vote | +2% |
| Security | Zero anomalies | +3% |

**Max Reward**: 60 EMO per block

## Staking

30% of validator balance earns staking rewards:
- **Liquid EMO**: 70% (available for transactions)
- **Staked EMO**: 30% (earning 12% APY)

Example:
```
Total: 1,000 EMO
Liquid: 700 EMO
Staked: 300 EMO
Monthly Reward: 3 EMO
Annual Reward: 36 EMO
```

2. **Slashing Penalties**: Malicious validator stake burned
3. **Token Buyback**: Foundation repurchases from market
